Installing Phase

Phase can be installed from the Pentaho marketplace. This is the recommended procedure if you simply want to use Phase to edit your Pentaho Anaylysis schemas.

You can also manually install phase. If you're a developer and you plan to make modifications to the Phase source code (or the documentation), you should use the manual installation procedure to test the changes you made.

Regardless of whether you choose to install Phase from the marketplace or manually, it is always a 2-step process:

  1. Install the Pentaho Database Inspection Service (Pedis). This is a content generator plugin which provides JDBC metadata for any JDBC connections available through Pentaho's data access plugin. Clients can obtain JDBC metadata (typically, in JSON format) through the REST calls handled by the content generator.
  2. Install Phase proper - the actual editor. The actual Pentaho Analysis Editor is a "thin client" - a pure client side application made up of HTML, CSS and javascript. It works by doing REST calls to the Pentaho Database Inspection Service, as well as a number of webservices provided by Pentaho's data access plugin.

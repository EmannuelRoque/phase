Manual Installation

Installing Pentaho plugins manually is typically not that hard. It requires you to download an archive containing all of the plugins executable code and resources, and unpacking that into the pentaho-solutions/system directory, which resides inside the home directory of your Pentaho Business Analytics server.

Manually installing Pedis

Find the pedis repository. on github. Switch to the desired branch and download the pedis.zip archive from the dist directory. Unzip pedis.zip into the pentaho-solutions/system directory. The archive should unpack itself into its own pedis directory.

Installing pedis requires a server restart. You can do that as soon as you're done installing plugins.

Manually installing Phase

Find the phase repository on github and locate the phase.zip archive in the dist directory. Unzip phase.zip into the pentaho-solutions/system directory. The archive should unpack itself into its own phase directory.

If Phase wasn't installed already you'll need to restart the Pentaho Business Analytics server. If you're only upgrading Phase it's enough to simply reload the Pentaho user console.

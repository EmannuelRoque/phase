Marketplace Installation

The marketplace is available by default in the community edition of the Pentaho Business Analytics Server.

If you're running the Enterprise edition of the Pentaho Business Analytics Server, you can install the marketplace plugin manually. Instructions to manually install the marketplace plugin can be found here.

If the marketplace plugin is installed, you can start the marketplace by going through the "Marketplace" perspective. You can enable the Marketplace perspective by choosing the Marketplace item in the perspectives menu. You can invoke this menu by clicking on the name of the current perspective, which is on the far left, directly beneath the menu bar.

Installing Pedis

Find the marketplace entry called "Pentaho Database Inspection Service" and click the install button. To quickly locate this marketplace entry, use the "search" field and type "pedis".

After installing Pedis, you'll get a notification along with a recommendation to restart the BI server. If you still need to install the Phase editor proper, don't restart just yet but install the Phase editor first. When both are installed, restart the server.

Installing Phase

Find the marketplace entry called "Pentaho Analysis Editor" and click the install button. To quickly locate this marketplace entry, use the "search" field and type "phase".

After installing Phase, you'll get a notification along with a recommendation to restart the BI server. If you still need to install the Pedis service, don't restart just yet but install the Pedis service first. When both are installed, restart the server.

If you're upgrading Phase, it is not strictly necessary to restart the Pentaho BI service. In this case, it is sufficient to refresh the Pentaho User Console in the browser. (If you can to be really sure, cler your browser cache as well.)

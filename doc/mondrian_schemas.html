Mondrian Schemas

Mondrian is a so-called Relational OLAP-engine (ROLAP). This means the actual data that makes up the measures and dimension data are sourced from a live relational database.

In order to perform its tasks, Mondrian needs to know how to connect to the relational database backend. It also needs to know how the tables and columns provided by the relational backend map to items in the relational model that mondrian presents. This mapping is defined in a so-called schema.

One mondrian schema is associated with one database connection but can define many different measures and dimensions. Mondrian schemas are typically expressed in specially designed XML-dialect.

Phase - Overview

Phase stands for Pentaho Analysis Editor.

Pentaho Analysis

Pentaho Analysis is a back-end component of the Pentaho Platform that provides Online Analytical Processing capabilities (OLAP). User-facing applications like Pentaho Analyzer and Saiku Analytics work on top of Pentaho Analysis to support the data analysis process. Other pillars of the Pentaho platform, like Pentaho reporting and dashboarding can also use Pentaho Analysis as a data source.

The Mondrian Engine

The heart of Pentaho Analysis is the Mondrian engine. Mondrian is a so-called Relational OLAP-engine (ROLAP). Being a ROLAP engine, Mondrian presents a multi-dimensional model suitable for analysis by business users on top of a relational database backend.

Mondrian Schemas

In order to perform its tasks, Mondrian needs to know how to connect to the relational database backend. It also needs to know how the tables and columns provided by the relational backend map to items in the multi-dimensional model that mondrian presents. This mapping between the relational database backend and the logical is defined in a so-called Mondrian schema or an Pentaho Analysis Model.

Where Phase fits in

Phase offers a graphical user interface that allows you to create and edit mondrian schemas. Phase is integrated with the Pentaho Business Analytics server. Phase runs directly as a web application or so-called thin client in the Pentaho user console.

Main Toolbar

The main toolbar is rectangular blue bar located at the very top of the phase application. It is always visible and contains a fixed set of buttons. You can hover your pointer over the toolbat buttons to reveal a tooltip that reveals the purpose of the button.

The buttons are described below:

Refresh

Pressing the refresh button reloads the Model Treeview.

New Schema Button

Pressing the "New Schema" button creates a new Mondrian schema (a new model). The new schema appears immediately in the Model Treeview. The treenode that is created to represent this new Schema will be automatically selected, which will automatically load the Schema Editor in the Editor area.

The new Schema is automatically given a name of the form "New Schema<n>", where <n> is some integer number.

Save Button

Pressing the Save button will save the local model to the Pentaho BI Server.

If this is a new schema it will simply be added. If this was an existing schema the old one will simply be overwritten.

If the current schema is an existing schema, but the name was modified, you will be prompted whether you want to create a copy of the existing schema with the new name, or if you want to rename the old schema. If there exists another schema with a name that is identical to the new name, you will be prompted to confirm whether you want to overwrite the existing schema.

Remove Button

Pressing the Remove Button will remove whatever model or model element is currently selected in the Model treeview. Since this is an irreverisble and destructive action, you will be prompted to confirm the removal.

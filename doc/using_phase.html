Using Phase

After installing phase you're ready to run it.

Running Phase

After Phase is installed, a new "Pentaho Analysis Editor" menu item will be added to the "Tools" menu. You can start Phase by clicking this menu item.

After clicking the "Pentaho Analysis Editor" menu item in the "Tools" menu, a new tab will open in the Pentaho User console, and the Phase user-interface will be loaded.
